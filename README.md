## 🚀 Deploy en GitHub Pages

1. Subí todos los archivos a un repositorio GitHub
2. Andá a **Settings → Pages**
3. En **Source** seleccioná `main` branch, carpeta `/root`
4. Guardá — en 1-2 minutos estará en `https://TU-USUARIO.github.io/TU-REPO`

## 📱 Instalar como app

**iPhone/iPad:**
1. Abrí la URL en Safari
2. Tocá el botón Compartir → "Agregar a pantalla de inicio"

**Android:**
1. Abrí la URL en Chrome
2. Tocá el menú → "Instalar app" o "Agregar a pantalla de inicio"

- Experiencias, restaurantes y platos típicos por destino
